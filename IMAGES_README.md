# Images Folder

This folder is for storing all local images used in the project.

## Required Images

Place the following images in this folder:

### Logo
- `logo.png` - Main logo (used in navigation, footer, and receipts)

### Event Images
- `event-music.jpg` - Music concert/event image
- `event-tech.jpg` - Technology conference image
- `event-food.jpg` - Food festival image
- `event-sports.jpg` - Sports event image
- `event-business.jpg` - Business event image
- `event-arts.jpg` - Arts & theater event image

## Image Specifications

- Recommended size for logo: 200x200px or similar aspect ratio
- Recommended size for event images: 400x250px (2:1 aspect ratio)
- Supported formats: PNG, JPG, JPEG, GIF, SVG

## Notes

- All images should be optimized for web use
- Use descriptive filenames (lowercase with hyphens)
- Ensure images have appropriate alt text for accessibility
